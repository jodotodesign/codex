# WordPress ACF Öffnungszeiten Plugin

Dieses Repository enthält das Plugin **ACF Öffnungszeiten & Feiertage** im Ordner [`wp-opening-hours-acf`](wp-opening-hours-acf/). Das Plugin legt im WordPress-Backend ACF-Felder für Öffnungszeiten, Bundesland, Brückentage und individuelle Schließtage an und stellt einen Shortcode bereit, der im Frontend anzeigt, ob aktuell geöffnet ist und wann wieder geöffnet wird.

Siehe die [Plugin-Dokumentation](wp-opening-hours-acf/README.md) für Installation und Nutzung.
